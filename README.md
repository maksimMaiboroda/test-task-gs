# XRM_Front
# RUN DEVELOPMENT VERSION
Prerequisites:
1. Node.js >= v8
2. npm     >= v6 

```javascript
1. npm install
2. npm run dev
3. open http://localhost:5173/ in browser
```

# BUILD PRODUCTION VERSION
Prerequisites:
1. Node.js >= v8
2. npm     >= v6 

```javascript
1. npm install
2. npm run build
```
Copy `public` folder to the server.
